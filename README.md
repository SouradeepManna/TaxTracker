# TaxTracker

TaxTracker is a full-stack Tax Filing and Management System built using Spring Boot, React, TypeScript, and MySQL.

## 🚀 Features

- JWT Authentication
- User Registration & Login
- Form 90C Tax Transaction Management
- Dashboard Insights
- Search & Filtering
- Export Reports (PDF, Excel, JSON)
- Swagger API Documentation

## 🛠️ Tech Stack

### Backend
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- Hibernate
- Maven

### Frontend
- React
- TypeScript
- Axios
- React Router

### Database
- MySQL

## ▶️ Run the Backend

```bash
mvn spring-boot:run
```

## ▶️ Run the Frontend

```bash
cd frontend
npm install
npm run dev
```
